# Barcade

A CLI for Barcade. Check Barcade's (Brooklyn) tap without leaving the comfy confines of the terminal.

![](http://i.imgur.com/UnQVPBo.png)

## Installation

    $ gem install barcade

## Usage

    $ barcade

There exists a `--yesterday` flag.

    $ barcade --yesterday
